*************************************************************************
***************** PARA CONSEGUIR RODAR O PROJETO ************************
*************************************************************************

1) Tenha instalado o Node.js, caso não tenha instalado,
https://nodejs.org/dist/v12.17.0/node-v12.17.0-x64.msi

2) E necessário o ambiente @angular/cli, caso não tenha instalado, 
digite o comando via terminal 'npm install -g @angular/cli'

3) Para rodar o projeto, acesse a pasta '/prova/prova-esfera/ 
via terminal e digite o comando 'ng serve'


-------------------------------------------------------------------------

*************************************************************************
********************** TESTES UNITÁRIOS *********************************

Para executar os teste

Acesse a pasta '/prova/prova-esfera/
