*************************************************************************
***************** PARA CONSEGUIR RODAR O PROJETO ************************
*************************************************************************

O projeto está disponível no github, no link abaixo.
https://github.com/Danilops-analistadeti/prova-teste.git

Caso tenha o Git CMD, pode se clonar o repositório utilizando o comando
'git clone https://github.com/Danilops-analistadeti/prova-teste.git'

--------------------------------------------------------------------------

1) Tenha instalado o Node.js, caso não tenha instalado, baixe pelo link
https://nodejs.org/dist/v12.17.0/node-v12.17.0-x64.msi

2) É necessário o ambiente @angular/cli, caso não tenha instalado, 
digite o comando via terminal 'npm install -g @angular/cli'

3) É necessário instalar as dependências, acesse a pasta '/prova-teste'
via terminal e digite o comando 'npm install' 

4) Para rodar o projeto, acesse a pasta '/prova-teste'
via terminal e digite o comando 'ng serve'

5) No navegador acesse a URL http://localhost:4200/

-------- BUILDAR PARA PRODUÇÃO -----------------------------------------

1) Acesse a pasta '/prova-teste' via terminal e digite 'ng build'

2) Em seguida '/prova-teste/dist/prova-teste' e digite 'ng serve'

3) No navegador acesse a URL http://localhost:4200/

-------------------------------------------------------------------------

*************************************************************************
********************** TESTES UNITÁRIOS *********************************
*************************************************************************

Para executar os teste

Acesse a pasta '/prova-teste' 
via terminal e digite os comandos: 
	
'ng lint'
'ng test'
'ng e2e'
